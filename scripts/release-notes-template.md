# Jot v1.13

> A smaller default surface, a clearer Setup Wizard, and richer feedback reports — without disturbing how you already use Jot.

## What's new

- **Home is now Recents.** The sidebar entry and pane title have been renamed; the pane and your stored recordings are unchanged.
- **Settings collapses in the sidebar.** Click the chevron next to Settings to fold General / Transcription / Vocabulary / Sound / AI / Shortcuts into a single row. State persists across launches.
- **Ask Jot follows your configured AI provider.** The separate "Allow Ask Jot to use this provider" toggle has been retired. If you explicitly opted out before upgrading, Ask Jot stays on Apple Intelligence for you — a one-time banner explains the change.
- **Setup Wizard defaults to Nemotron with one click.** The model step shows just Nemotron pre-selected with a Download button; "Show 3 more options" reveals Parakeet v3 multilingual + EOU, Parakeet Japanese, and Parakeet v2 (deprecated) if you want them.
- **Setup Wizard hotkey step is redesigned.** One focal chip showing your bound key plus three quick-pick alternatives (Caps Lock · ⌥ Right Option · ⌥ Space) and a Custom… recorder for chord combos. No more separate "Trigger type" picker — the wizard infers chord vs single-key from what you tap. The Input Monitoring permission banner now surfaces immediately on page load instead of after a 12-second silent timer.
- **Send Feedback now supports screenshot attachments.** Attach up to 3 images via the paperclip; Jot auto-shrinks them to fit a 5 MB cap. The redacted log and app-details footer are still pre-filled, and "Show original log" is still a one-click toggle.
- **New "Show advanced features" toggle in Settings → General.** If you ever want a slimmer Jot, turn it off to hide Custom Vocabulary, Ask Jot, Push-to-Talk, and Paste Last Result. Upgrading users start with this **on**, so nothing changes by default.

## Fixes

- **Speaker Labels card no longer leaks into Transcription settings.** The hidden feature's kill switch is now respected end-to-end.
