# Durable distribution channels

These are the compounding channels — unlike social posts, they keep delivering
downloads months after the one-time setup. Work them in this order:

| # | Channel | Kit | Effort | What it does |
|---|---------|-----|--------|--------------|
| 1 | Homebrew tap | [homebrew.md](homebrew.md) | 10 min | `brew install --cask …/jot` — the install path the HN/dev crowd trusts; goes on README + website |
| 2 | AlternativeTo | [alternativeto.md](alternativeto.md) | 20 min | Catches everyone Googling "superwhisper free alternative" — highest-intent traffic there is |
| 3 | MacUpdate | [macupdate.md](macupdate.md) | 20 min | Directory + Google results for "Jot mac download" |
| 4 | Product Hunt | [product-hunt.md](product-hunt.md) | half day | One-time spike (300–1500 visits) + permanent Google-ranked listing. Gated: launch only when demo video + analytics + brew are live |

Also worth doing when convenient (no kit needed — open a PR adding one line):
- **awesome-mac** (github.com/jaywcjlove/awesome-mac) — Dictation/Audio section
- **Open-source Mac apps lists** (e.g. github.com/serhii-londar/open-source-mac-os-apps)

Status is tracked in Mission Control's "Durable channels" panel.
The daily marketing agent handles the social layer; this folder is the
foundation layer underneath it.
