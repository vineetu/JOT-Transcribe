**Full description:**
Jot is a dictation utility for macOS. Press a global hotkey, speak, and the
transcript is pasted wherever your cursor is — email, Slack, your code editor,
any app.

Transcription runs entirely on-device on the Apple Neural Engine (NVIDIA's
Parakeet model via Core ML), so it works offline and audio never leaves the
Mac. There is no account, no subscription, and no telemetry. Optional AI
cleanup and rewrite-by-voice default to Apple Intelligence on-device, with
local Ollama or your own API keys (OpenAI, Anthropic, Gemini) as alternatives.

Other features: custom vocabulary, rewrite selected text with a spoken
instruction, searchable transcription history, menu-bar quick access, and a
built-in help chatbot. Free and open source (MIT). Requires Apple Silicon and
macOS 14+.
